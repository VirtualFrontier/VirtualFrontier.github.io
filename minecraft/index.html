The Virtual Frontier X Minecraft

Pourquoi ce serveur minecraft ?

Virtual Frontier X Minecraft est un serveur minecraft de la communauté de Virtual Frontier, il vous permet de vous connecter sur n'importe quelle plateforme et de jouer avec vos amis et notre communauté.

Quelques informations importantes sur le serveur minecraft :

  • Java Edition, l'ip est 109.24.202.160, et le port est 25565

  • Bedrock Edition, l'ip est 109.24.202.160, et le port est 19132
